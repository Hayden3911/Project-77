In programming, a console is a text-based interface that allows you to interact with the computer's operating system or run code. With the console, you can write and execute commands or programs. It is also used for debugging and troubleshooting. The open the console screen, inspect the web page you are on and switch the tabs at the top of the new screen from elements to console. To print things on the console, use JavaScript and console.log(yourvalue).
